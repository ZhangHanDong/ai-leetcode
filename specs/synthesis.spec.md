# Cross-Problem Synthesis Spec

## Goal

从多道已验证题目中提炼稳定模式，而不是汇总若干题解摘要。

## Minimum evidence

- 至少三道已达到 `verified` 的题；
- 至少一个共享不变量；
- 至少一个约束变化导致的变体；
- 至少一个模式不适用的反例；
- 所有关系边都有文字理由。

## Required chapter structure

1. 暴力解中重复工作的共同形态；
2. 模式的状态、不变量和转移；
3. 三道题分别新增了什么理解；
4. 通用骨架及可变插槽；
5. 前提、失效点和替代方案；
6. 迁移练习。

## Acceptance criteria

- 不重复粘贴各题完整分析；
- 每个案例承担不同教学作用；
- 模式 ID、关系类型与题目元数据一致；
- 模式页面能够反向链接全部代表题目。
