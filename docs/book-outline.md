# 全书大纲

## 定位

这不是按题号排列的答案集，而是一本从题目中抽取算法模型、状态、不变量、证明和工程权衡的书。

## 阅读路径

### 路径 A：建立算法思维

前言 → 建模 → 状态与不变量 → 数组与窗口 → 树与图 → 动态规划 → 贪心证明

### 路径 B：面试复习

模式索引 → 对应模式章节 → 典型题深挖 → 易错边界 → 迁移练习

### 路径 C：工程与性能

复杂度模型 → 数据结构布局 → benchmark 方法 → 多实现比较 → 执行沙箱

## 分篇规划

1. **算法问题的建模方法**：约束、目标、输入规模、状态空间和问题变换。
2. **状态、边界与不变量**：循环不变量、搜索边界、递归契约和正确性证明。
3. **数组、字符串与指针技巧**：双指针、前缀量、差分、滑动窗口。
4. **栈、队列与单调结构**：单调栈、单调队列、括号结构、下一个更大元素。
5. **树、图与连通性**：DFS、BFS、拓扑序、并查集、最短路。
6. **搜索、剪枝与回溯**：选择空间、恢复现场、上下界和记忆化。
7. **动态规划**：状态定义、转移顺序、空间压缩、区间与树形 DP。
8. **贪心及其证明**：交换论证、局部最优条件、反例与边界。
9. **算法工程与性能**：数据布局、分配、缓存、常数项和可复现 benchmark。
10. **综合题族与迁移训练**：跨题共性、条件变化、算法失效点和组合模式。

## 每篇最低完成标准

- 至少一个核心模式章节；
- 至少三道相互关联的题；
- 至少一条跨题 E2E 推导；
- 至少一个可交互 Trace；
- 至少一组多实现对比；
- 明确本篇模式不适用的反例。
